コンテンツ
==============================================
こちらのページでは、以下の内容をご紹介しております。

- 本セットアップガイドにて、F5 Advanced Firewall Manager（以下、AFM）のポリシーの設定方法についてご案内します。
- AFMは、ファイアウォールや、DDoS対策、IPS機能を提供します。
- 本ガイドでは、AFMをご購入いただいてすぐに主要機能を導入頂けるように、必要となる典型的なセットアップ手法を、豊富なスクリーンショットを交えて解説します。（実際は環境構成にあった設定値を設定して下さい。）
- 本ガイドでは、F5 Japanにおけるハンズオントレーニングの標準コース外になりますので、担当営業・SEに個別にご相談ください。
